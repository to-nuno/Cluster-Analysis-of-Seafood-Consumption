## **3.データと方法**
### 使用データ
使用するデータは以下の通りである。
- 都道府県庁所在地別1世帯あたりの年間の品目別支出金額(2人以上の世帯)
- データの出典は、総務省統計局の家計調査年報(家計収支編)のうち、2人以上の世帯における魚介類の都市階級・地方・都道府県庁所在市別(支出金額及び購入数量のみ)

### 対象期間
- 分析の対象期間は、2020〜2024年の間とする。
- 対象期間の設定理由は、以下のとおり
 - 利用可能なデータの最新年が2024年であること。
 - 2020年はCOVID19の感染拡大の影響で消費動向が変化している可能性があるため、2020年以降とする。

#### データ処理
1. エクセルファイルの前処理
- Pythonのpandasを用いて、エクセルファイルの統計表に含まれる余分な行と列を削除する　　※政府の公開されている統計表には、説明のための余計な列や行が含まれており、そのままでは分析にかけることができないことがほとんど
2. データの実質化と標準化
- 実質化
 - 前処理を行ったデータに対して、小売物価統計調査の10大費目別消費者物価地域差指数の食料の指数で除す(指数は都道府県庁所在地別のものを使用する)。
 - この処理を行うことで、地域間の物価の違いを取り除き、データを実質化する。
- 標準化
 - 実質化したデータを、品目別に標準化する。
 - 例えば、マグロの都道府県庁所在地別のデータで標本平均及び標本標準偏差(n-1で除したもの)を計算し、その値を使用してマグロの各都道府県庁所在地のデータを標準化する。
